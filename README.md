# Portfólio Profissional - Desenvolvimento Web

Este projeto é um portfólio pessoal e acadêmico desenvolvido como parte das atividades práticas do curso de **Sistemas de Informação**. O site foi construído com foco em design responsivo e arquitetura CSS limpa, apresentando trajetórias acadêmicas, habilidades técnicas e projetos de forma profissional.

## 📝 Descrição do Projeto
O site é um portfólio estático multipáginas que utiliza o framework Bootstrap 5.3 para garantir a adaptabilidade em diferentes tamanhos de tela (mobile-first). O design prioriza o minimalismo, utilizando uma paleta de cores baseada em tons de azul e branco para transmitir seriedade e clareza.

### Principais Seções:
* **Início:** Apresentação com avatar estilizado e alinhamento responsivo.
* **Sobre mim:** Detalhamento da trajetória acadêmica e profissional.
* **Habilidades:** Vitrine interativa com carrosséis categorizados 
* **Cursos:** Exibição da formação acadêmica (UPE) e técnica (Redes de Computadores) em cards personalizados.
* **Contato:** Formulário de contato com validação visual e foco estilizado.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica.
* **CSS3 Personalizado:** * Uso de **Variáveis Globais** (`:root`) para facilitar a manutenção.
* **Bootstrap 5.3:** Sistema de Grid, Componentes e Utilitários de espaçamento responsivo.
* **JavaScript:** Lógica de interatividade para componentes móveis e carrosséis.
